# special_method
My special method project
_add_() – Defines how the addition (+) operator behaves for objects of a class.
_sub_() – Defines how the subtraction (-) operator behaves for objects of a class.
_mul_() – Defines how the multiplication (*) operator behaves for objects of a class.
_getitem_() – Allows an object to be accessed using indexing, such as object[index].
_setitem_() – Allows values in an object to be modified using indexing, such as object[index] = value.
_contains_() – Defines how the in operator checks whether an item exists within an object.
_call_() – Allows an object to be called like a function using parentheses.
_iter_() – Returns an iterator object, making the object usable in a for loop.
_next_() – Returns the next item in an iteration and raises StopIteration when there are no more items.
_bool_() – Determines whether an object evaluates to True or False in Boolean expressions.
_del_() – Called automatically when an object is about to be destroyed, allowing cleanup of resources if necessary
